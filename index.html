Sekilas Materi

Fungsi Pertumbuhan

Grafik monoton naik jika nilai basis a > 1. Contoh: f(x) = 2x

Fungsi Peluruhan

Grafik monoton turun jika nilai basis 0 < a < 1. Contoh: f(x) = (1/2)x

Materi Fungsi Eksponensial

Bentuk Umum Fungsi Eksponensial

Fungsi eksponensial adalah pemetaan bilangan real x ke bilangan a dengan bentuk umum:

f(x) = k · ax

Syarat penting:

  • ✅ a adalah basis atau bilangan pokok, dengan syarat a > 0 dan a ≠ 1.
  • ✅ x adalah eksponen atau pangkat (merupakan domain/daerah asal).
  • ✅ k adalah konstanta sembarang (k ≠ 0).

Langkah-langkah Menggambar Grafik

Untuk menggambar grafik fungsi eksponensial, ikuti 3 langkah mudah berikut:

  1. Buat Tabel Titik Bantu: Pilih beberapa nilai x (misalnya dari -2 sampai 2), lalu hitung nilai y = f(x) yang bersesuaian.
  2. Letakkan Titik Koordinat: Gambar sumbu x dan sumbu y (bidang Kartesius), kemudian tandai titik-titik (x, y) yang didapat dari tabel.
  3. Hubungkan Menjadi Kurva: Hubungkan titik-titik tersebut dengan kurva mulus. Ingat, kurva tidak boleh menyentuh asimtot!

Unsur dan Sifat Grafik

Grafik fungsi eksponensial f(x) = ax memiliki sifat-sifat unik:

  • 🔹 Memotong Sumbu Y: Selalu memotong sumbu Y di titik (0, 1) jika k=1.
  • 🔹 Asimtot Datar: Kurva akan terus mendekati sumbu X (y=0) tetapi tidak akan pernah menyentuhnya.
  • 🔹 Monoton Naik: Jika basis a > 1, grafik akan terus naik dari kiri ke kanan.
  • 🔹 Monoton Turun: Jika basis 0 < a < 1, grafik akan terus turun dari kiri ke kanan.

Contoh Soal

Mari kita pelajari contoh penyelesaian soal berikut.

Contoh 1: Menggambar f(x) = 2x

Langkah: Buat tabel untuk x = -1, 0, 1, 2

x-1012
y1/2124

Titik koordinat: (-1, 0.5), (0, 1), (1, 2), (2, 4). Hubungkan titik tersebut menjadi kurva naik.

Contoh 2: Menggambar f(x) = (1/2)x

Langkah: Buat tabel untuk x = -2, -1, 0, 1

x-2-101
y4211/2

Titik koordinat: (-2, 4), (-1, 2), (0, 1), (1, 0.5). Hubungkan titik tersebut menjadi kurva turun.

Contoh 3: Titik Potong

Soal: Tentukan titik potong grafik f(x) = 3x + 2 terhadap sumbu Y!

Jawab:
Memotong sumbu Y berarti nilai x = 0.
y = 30 + 2
y = 1 + 2 = 3
Jadi, titik potongnya adalah (0, 3).
